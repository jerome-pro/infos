====== PSEUDO CLASSES & PSEUDO ELEMENTS ======

===== PSEUDO CLASSES =====

Ce sont les sélecteurs qui commencent par "** : **"

==== Sélecteurs utiles ====

__pour les liens :__
<file>
:hover / :focus / :active / :visited
</file>

__en fonction de la position :__
<file>
:first-child / :last-child / :only-child

:nth-child() / :nth-of-type()
</file>
